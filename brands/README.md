# Home Assistant Brands PR Bundle

This folder contains the branding assets for the custom integration domain:
- `mybag_aero_tracker`

Target path in `home-assistant/brands`:
- `custom_integrations/mybag_aero_tracker/icon.png`
- `custom_integrations/mybag_aero_tracker/icon@2x.png`

## Submit upstream
1. Fork `https://github.com/home-assistant/brands`
2. Copy only icon files from this repo into your fork under `custom_integrations/mybag_aero_tracker/`:
   - `icon.png`
   - `icon@2x.png`
3. Open PR to `home-assistant/brands`
4. After merge, restart Home Assistant and refresh browser cache.
